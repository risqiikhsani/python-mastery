## How to Use

Each folder contains Python files covering subtopics. Run any file directly or use `main.py` to switch between topics.

**Option 1: Run individual files directly**
```bash
python3 oop/classes_objects.py
```

**Option 2: Use main.py (uncomment to switch files)**
1. Open `main.py`
2. Uncomment the import you want to run
3. Run `python3 main.py`

```python
# In main.py, uncomment the file you want:
from oop.classes_objects import *
```
